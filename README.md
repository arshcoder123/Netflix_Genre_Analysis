# Netflix Genre Analysis

A Data Analysis project focused on exploring Netflix content trends, genre popularity, ratings, and content distribution using Python and Power BI.

---

## 📌 Project Overview

This project analyzes Netflix movies and TV shows to uncover insights related to:

- Most popular genres
- Content growth over the years
- Ratings distribution
- Movies vs TV Shows
- Country-wise content production
- Duration trends
- Netflix content strategy

The project demonstrates real-world data cleaning, exploratory data analysis (EDA), and dashboard building skills for Data Analytics portfolios.

# 📊 Key Insights

- Drama and Comedy are among the most common genres
- Netflix content increased rapidly after 2015
- Movies dominate the platform compared to TV shows
- Certain countries contribute significantly more content
- TV-MA is one of the most frequent ratings
